# Portfolio Site Instructions

This is a GitHub Pages portfolio for a Roblox/game programmer.

Use:
- Plain HTML
- Plain CSS
- Minimal JavaScript
- No React
- No TypeScript
- No backend
- No package manager
- No build step unless explicitly requested

Style:
- Clean
- Dark theme
- Professional
- Fast-loading
- Mobile responsive

Content rules:
- Do not invent client names, brand names, metrics, awards, or project details.
- Use placeholder text where real project details are missing.
- Prefer "confidential client project" when details are unknown.
- Emphasize Roblox programming, gameplay systems, backend systems, UI, economy/progression, tools, live-service features, and performance.

File structure:
- index.html
- style.css
- script.js
- assets/images/
- assets/gifs/
- projects/

When changing code:
- Keep it readable.
- Avoid unnecessary dependencies.
- Keep GitHub Pages compatibility.
- Use relative links.
